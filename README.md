## About

An application that allows you to check whether the network is reachable using the CLI.

## Installation

```
$ cargo install --path . && ipsc
```

## Options

- [`arp`]: run arp
- [`ping`]: run ping
- [`can_communication`]: run test communication
- [`show_macaddress`]: run shows macaddress
